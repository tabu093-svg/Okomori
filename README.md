### 🔥御神火の片付け
- 御神火の消化
- 御神火の解体
    - 燃え残りを**重機**で解体して、2tトラックに積む
        - 積んだ後、仮置き場まで運搬
    - 解体時に、鎹を回収する(xx本)
    - 消防ポンプ、ホースの片づけ

### ⛺️振る舞い周辺の片付け
 
- 物品の片づけ
    - テント/椅子
        - 市役所から借用したものは番号が  
          振ってあるので番号通りに片付け
    - ハソリ
    - 飾り付け
    - ゴミ/残飯
- 物品の運搬
    - 御神火解体が完了し、導線が確保できたら
      オクトピアへ運搬
    - 太太神楽式用の分は倉庫へ運搬
      **机(脚長)12台／机(脚短)2台／椅子15脚**


### 💡照明の片付け（タップで開く
 
- 照明を回収、収納箱に入れる
    - **暗く、高い所にも設置されているので気を付けて作業**
